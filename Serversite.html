PHP

PHP er et udbredt sprog når man snakker webudvikling og koder til dynamiske websider.

PHP er et rekursiv akronym som står for "Hypertext Preprocessor" og ligner meget Javascript i sin opbygning. Dette betyder at PHP køre samme princip som Java i forhold til at PHP køres fra en server for derefter at sendes tilbage Html til brugeren - hvor Java i stedet sender direkte til brugerens browser.

Igennem PHP er det blandt andet muligt at sende emails, lave afstemninger og have loginfunktioner på hjemmesiden.

Javascript

En scripting-metode som skrives direkte ind i html-filerne. Ved at indsætte javascript giver man komandoen til browseren som efterfølgende læser koden og udføre den.

Javascript er blandt andet med til at man kan forstørre billeder uden af forlade den pågændende side man befinder sig på.
